**Les utilisateurs : abonnés et non-abonnés On remarque qu’il y a 4 catégories d’utilisateurs alors qu’on en distingue que deux : les abonnés : “member” et “Subscriber” les utilisateurs occasionnels : “Customer” et “casual” On va donc remplacer “Subscriber” par “member”, puis “Customer” par “casual” pour n’avoir que deux valeurs uniques: “member”: pour les utilisateurs avec un abonnement annuel * “casual”: pour les utilisateurs ponctuels sans abonnement

all_trips <- all_trips %>% mutate(member_casual= recode(member_casual,

***Les dates Nous allons travailler sur la variable “started_at”, afin d’obtenir les éléments suivants : - l’année : “year” - le mois : “month” - le jour du trajet: “day_of_week” - la durée d’un trajet : “ride_length”, en secondes, qui sera calculer en soustrayant l’heure de “ended_at” à l’heure de “started_at”.

all_trips$date <- as.Date(all_trips$started_at) #The default format is yyyy-mm-dd
all_trips$month <- format(as.Date(all_trips$date), "%B") # %B :Pour avoir les mois en toute lettre
all_trips$day <- format(as.Date(all_trips$date), "%d")
all_trips$year <- format(as.Date(all_trips$date), "%Y")
all_trips$day_of_week <- format(as.Date(all_trips$date), "%A") # Pour avoir le jour de la semaine

# Add a "ride_length" calculation to all_trips (in seconds)
all_trips$ride_length <- difftime(all_trips$ended_at,all_trips$started_at) #on a un nouveau variable de type difftime
